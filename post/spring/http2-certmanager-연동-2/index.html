к°ңмҡ”

мқҙм „ л¬ём„ңм—җм„ң мҝ лІ„л„ӨнӢ°мҠӨ нҷҳкІҪм—җ cert-managerлҘј м„Өм№ҳн•ҙм„ң нҷҳкІҪкө¬м„ұмқ„ мҷ„лЈҢн–ҲлӢӨл©ҙ, мқҙлІҲ л¬ём„ңм—җм„ңлҠ” spring boot нҷҳкІҪ кө¬м„ұ л°Ҹ н…ҢмҠӨнҠёк№Ңм§Җ м•Ңм•„ ліј мҲҳ мһҲлҸ„лЎқ н•©лӢҲлӢӨ.



мӨҖл№„

http2 н…ҢмҠӨнҠёлҘј мң„н•ҙм„  лӢӨмқҢкіј к°ҷмқҖ н…ҢмҠӨнҠё нҷҳкІҪмқҙ н•„мҡ”н•©лӢҲлӢӨ.

  • key, crt нҢҢмқј
    • н•ҙлӢ№ нҢҢмқјмқҖ мқҙм „ л¬ём„ңм—җм„ң м§„н–ү нӣ„ мғқм„ұлҗң secretsм—җм„ң м¶”м¶ңн•©лӢҲлӢӨ.
  • н…ҢмҠӨнҠёмҡ© API м„ңлІ„
    • Spring BootлЎң к°„лӢЁн•ҳкІҢ нҷҳкІҪкө¬м„ұ н•©лӢҲлӢӨ.
    • kubernetes нҷҳкІҪм—җ л°°нҸ¬лҘј мң„н•ҙ Dockerfile кіј deployment, service yaml к№Ңм§Җ мӨҖл№„н•©лӢҲлӢӨ.

spring boot м„ңлІ„ кё°ліё кө¬м„ұмқҖ м•„лһҳмҷҖ к°ҷмқҙ м§„н–үн–ҲмҠөлӢҲлӢӨ.

лЎңм»¬ н…ҢмҠӨнҠё

мқёмҰқнҢҢмқј м¶”м¶ң

cert-managerлҘј нҶөн•ҙ мғқм„ұлҗң secrets нҢҢмқјмқ„ ліҙл©ҙ ca.crt, tls.crt, tls.key нҢҢмқјмқҙ мһҲмҠөлӢҲлӢӨ. мқҙ мӨ‘, tls.crt, tls.key нҢҢмқјмқ„ м¶”м¶ңн•ҙліҙлҸ„лЎқ н•©лӢҲлӢӨ.

secrets нҢҢмқјмқ„ json нҳ•мӢқмңјлЎң мқҪмқҖ л’Ө, data.tls.key(crt)лҘј к°Җм ёмҳЁ лӢӨмқҢ, мқёмҪ”л”©лҗң base64лҘј л””мҪ”л”© н•ң лӢӨмқҢ, нҢҢмқјлЎң л§Ңл“ңлҠ” лӘ…л №м–ҙ мһ…лӢҲлӢӨ.

# tls.key
kubectl get secrets selfsigned-cert-tls -o jsonpath='{.data.tls\.key}' | base64 --decode >> tls.key
# tls.crt
kubectl get secrets selfsigned-cert-tls -o jsonpath='{.data.tls\.crt}' | base64 --decode >> tls.crt

м¶”м¶ңлҗң нҢҢмқјмқҖ spring bootлЎң л§Ңл“  м„ңлІ„ мӨ‘ мӣҗн•ҳлҠ” кІҪлЎң лҳҗлҠ” кҙҖлҰ¬н•ҳлҠ” кІҪлЎңм—җ мң„м№ҳмӢңмјң мЈјлҸ„лЎқ н•©лӢҲлӢӨ.
мқҙлІҲ л¬ём„ңм—җм„ңлҠ” classpath кІҪлЎңмқё src/main/resources н•ҳмң„м—җ мң„м№ҳмӢңнӮӨлҸ„лЎқ н•©лӢҲлӢӨ.



application.yaml

http2 м„Өм •мқ„ мң„н•ҙ application.yamlм—җ м„Өм •мқ„ м¶”к°Җ л°Ҹ tls.key, tls.crt нҢҢмқјмқҳ кІҪлЎңлҘј мһЎм•„мӨҳм•ј н•©лӢҲлӢӨ.

server:
  ssl:
    enabled: true
    certificate: classpath:tls.crt
    certificate-private-key: classpath:tls.key
  http2:
    enabled: true



н…ҢмҠӨнҠё м§„н–ү

м„ңлІ„лҘј мӢӨн–ү н•ҳкі лӮҳм„ң, лёҢлқјмҡ°м Җм—җм„ң APIлҘј нҳём¶ң н•ҳл©ҙ лӢӨмқҢкіј к°ҷмқҖ мӮ¬мӢӨмқ„ м•Ңм•„лӮј мҲҳ мһҲмҠөлӢҲлӢӨ.

  • https мӢӨн–ү
  • h2(http2) н”„лЎңнҶ мҪң мӢӨн–ү




мҝ лІ„л„ӨнӢ°мҠӨ н…ҢмҠӨнҠё

Dockerfile мғқм„ұ

мҝ лІ„л„ӨнӢ°мҠӨ л°°нҸ¬лҘј мң„н•ҙ лЁјм Җ м»Ён…Ңмқҙл„Ҳ мқҙлҜём§ҖлҘј л§Ңл“Ө мҲҳ мһҲлҸ„лЎқ DockerfileлҘј мһ‘м„ұн•©лӢҲлӢӨ.
build/libs н•ҳмң„м—җ мһҲлҠ” jarлҘј нҢҢмқјмқ„ м°ҫм•„ app.jar нҢҢмқјлЎң ліөмӮ¬н•ҳм—¬ мӢӨн–үн•  мҲҳ мһҲлҸ„лЎқ н•©лӢҲлӢӨ.

FROM eclipse-temurin:11.0.16.1_1-jdk
ENV TZ=Asia/Seoul
#COPY ./target/*.jar app.jar
COPY ./build/libs/*.jar app.jar
ENTRYPOINT ["java", "-jar", "/app.jar"]

jar мғқм„ұ

Dockerfileмқ„ нҶөн•ҙ мқҙлҜём§Җм—җ jarнҢҢмқјмқ„ ліөмӮ¬н•ҳкё° мң„н•ҙ лҸҷмһ‘н•  мҲҳ мһҲлҠ” jarнҢҢмқјмқ„ мғқм„ұн•ҙм•ј н•©лӢҲлӢӨ.
gradle кё°мӨҖмңјлЎң лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

gradle clean bootJar



image л№Ңл“ң

Dockerfileкіј jarнҢҢмқјк№Ңм§Җ мӨҖл№„ лҗҳм—Ҳмңјл©ҙ, мқҙлҜём§ҖлҘј л№Ңл“ң н•  мҲҳ мһҲлҸ„лЎқ н•©лӢҲлӢӨ.

docker build -t http2-test:0.0.1 .



deployment, service yaml мһ‘м„ұ

мҝ лІ„л„ӨнӢ°мҠӨм—җ л°°нҸ¬н•ҳкё° мң„н•ң deploymentмҷҖ л°°нҸ¬лҗң podм—җ м ‘к·јн•ҳкё° мң„н•ң serviceлҘј мғқм„ұн•  мҲҳ мһҲлҠ” yamlмқ„ мһ‘м„ұн•©лӢҲлӢӨ.

deployment.yaml

yaml мһ‘м„ұ мӢң мқҙм „ л¬ём„ңм—җм„ң м§„н–үлҗң cert-managerлҘј нҶөн•ҙ л§Ңл“Өм–ҙм§„ secretsлҘј volumeMountн•ҳм—¬ м»Ён…Ңмқҙл„Ҳ лӮҙл¶Җм—җ л§ҲмҡҙнҠё н•  мҲҳ мһҲлҸ„лЎқ мһ‘м„ұн•©лӢҲлӢӨ. л§ҲмҡҙнҠёлҗң нҢҢмқјмқҖ нҷҳкІҪліҖмҲҳлҘј нҶөн•ҙм„ң л§Өн•‘н•  мҲҳ мһҲлҸ„лЎқ н•©лӢҲлӢӨ.

apiVersion: apps/v1
kind: Deployment
metadata:
  name: http2-test-deployment
  labels:
    app.kubernetes.io/name: spring
spec:
  replicas: 1
  selector:
    matchLabels:
      app.kubernetes.io/name: spring
  template:
    metadata:
      labels:
        app.kubernetes.io/name: spring
    spec:
      containers:
        - name: http2-test
          # лЎңм»¬м—җм„ң л№Ңл“ңн•ң мқҙлҜём§ҖлҘј мһ‘м„ұн•ҳл©ҙ лҗңлӢӨ.
          image: http2-test:1.0.0
          imagePullPolicy: IfNotPresent
          ports:
            - containerPort: 8080
              name: spring-port
          # мқёмҰқм„ң нҢҢмқјмқ„ л§Өн•‘н•ҳлҠ” нҷҳкІҪліҖмҲҳ
          env:
            - name: SERVER_SSL_CERTIFICATE
              value: /etc/secret-volume/tls.crt
            - name: SERVER_SSL_CERTIFICATE-PRIVATE-KEY
              value: /etc/secret-volume/tls.key
          # secretsм—җ мһҲлҠ” мқёмҰқм„ңлҘј нҠ№м • л””л үнҶ лҰ¬м—җ л§ҲмҡҙнҠё
          volumeMounts:
            - name: tls-secret-volume
              mountPath: /etc/secret-volume
      # secretsм—җ мһҲлҠ” мқёмҰқм„ңлҘј нҠ№м • л””л үнҶ лҰ¬м—җ л§ҲмҡҙнҠён•ҳкё° мң„н•ҙ ліјлҘЁ м„Өм •
      volumes:
        - name: tls-secret-volume
          secret:
            # cert-managerм—җ мқёмҰқм„ң м„Өм •н•ҳл©ҙм„ң мғқкёҙ secretsлӘ…
            secretName: selfsigned-cert-tls

мһ‘м„ұ мҷ„лЈҢлҗң yamlмқҖ л°°нҸ¬н•  мҲҳ мһҲлҸ„лЎқ н•©лӢҲлӢӨ.

kubectl apply -f deployment.yaml

service.yaml

deployment.yamlм—җм„ң мһ‘м„ұн•ң container portмҷҖ label selectorлҘј мһҳ л§һм¶°м„ң мһ‘м„ұн•ҙмЈјлҸ„лЎқ н•©лӢҲлӢӨ.
NodePortлЎң нғҖмһ…мқ„ м§Җм •н•ҳм—¬ podм—җ м ‘к·ј н•  мҲҳ мһҲлҸ„лЎқ н•©лӢҲлӢӨ.

apiVersion: v1
kind: Service
metadata:
  name: http2-test-service
spec:
  type: NodePort
  selector:
    app.kubernetes.io/name: spring
  ports:
    - protocol: TCP
      port: 8080
      targetPort: spring-port
      name: http2-test-port

мһ‘м„ұ мҷ„лЈҢлҗң yamlмқҖ л°°нҸ¬н•  мҲҳ мһҲлҸ„лЎқ н•©лӢҲлӢӨ.

kubectl apply -f service.yaml

н…ҢмҠӨнҠё

м •мғҒм ҒмңјлЎң л°°нҸ¬к°Җ мҷ„лЈҢ лҗҳм—ҲлӢӨл©ҙ лӢӨмқҢкіј к°ҷмқҙ мӢӨн–үн•ҳлҠ” лӘЁмҠөмқ„ нҷ•мқён•  мҲҳ мһҲмҠөлӢҲлӢӨ.

мқҙм ң, NodePortлЎң мғқм„ұлҗң portлҘј нҶөн•ҙ м ‘к·јн•ҳм—¬, httpsмҷҖ http2 н”„лЎңнҶ мҪңмқҙ м •мғҒм ҒмңјлЎң м Ғмҡ©лҗҳм—ҲлҠ”м§Җ нҷ•мқён•  мҲҳ мһҲлҸ„лЎқ н•©лӢҲлӢӨ.

м •мғҒм ҒмңјлЎң лҸҷмһ‘н•ҳлҠ” кІғмқ„ нҷ•мқён•  мҲҳ мһҲмҠөлӢҲлӢӨ.
лҳҗн•ң, deployment.yamlм—җм„ң volumeMount мқҙмҡ©н•ҳм—¬ мқёмҰқнҢҢмқјмқ„ л§ҲмҡҙнҠё н•ң кІғмқҖ лӢӨмқҢкіј к°ҷмқҙ podм—җ м§Ғм ‘м ҒмңјлЎң м ‘к·јн•ҳм—¬ нҷ•мқён•  мҲҳ мһҲмҠөлӢҲлӢӨ.

# pod лӮҙл¶Җм—җм„ң
